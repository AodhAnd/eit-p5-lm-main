46 - Accuracy
47  - 8 bit timers (0,2) 1 millisecond is 15.625 countdown on timer
48  - example 10 msec 156 instead of 156.25 so an error of 0.25/156.25 ~= 0.2%
49  - 16 bit timers count down is 1 millisecond for 62.5 count

55 Timer0:
56 - Timer0 and 2 is a 8bit timer.
57 - In the Arduino world Timer0 is been used for the timer functions, like delay(), millis() and micros().
58 - If you change Timer0 registers, this may influence the Arduino timer function.
59 - So you should know what you are doing.
60 
61 Timer1:
62 - Timer1 is a 16bit timer.
63 - In the Arduino world the Servo library uses Timer1 on Arduino Uno (Timer5 on Arduino Mega).
64 
65 Timer2:
66 - Timer2 is a 8bit timer like Timer0.
67 - In the Arduino work the tone() function uses Timer2.
68 
69 Timer3, Timer4, Timer5: Timer 3,4,5 are only available on Arduino Mega boards.
70 - These timers are all 16bit timers.

94 - Servo Library uses Timer1.
95 - - You can’t use PWM on Pin 9, 10 when you use the Servo Library on an Arduino.
96 - - For Arduino Mega it is a bit more difficult. The timer needed depends on the number of servos.
97 - - Each timer can handle 12 servos.
98 - - For the first 12 servos timer 5 will be used (losing PWM on Pin 44,45,46).
99 - - For 24 Servos timer 5 and 1 will be used (losing PWM on Pin 11,12,44,45,46)..
100 - - For 36 servos timer 5, 1 and 3 will be used (losing PWM on Pin 2,3,5,11,12,44,45,46)..
101 - - For 48 servos all 16bit timers 5,1,3 and 4 will be used (losing all PWM pins).
102 
103 - Pin 11 has shared functionality PWM and MOSI.
104 - - MOSI is needed for the SPI interface, You can’t use PWM on Pin 11 and the SPI interface at the same time on Arduino.
105 - - On the Arduino Mega the SPI pins are on different pins.
106 
107 - tone() function uses at least timer2.
108 - - You can’t use PWM on Pin 3,11 when you use the tone() function an Arduino and Pin 9,10 on Arduino Mega.
